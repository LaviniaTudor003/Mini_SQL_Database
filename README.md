# Mini_SQL_Database
Acest proiect oferă o viziune funcțională asupra modului în care putem interacționa cu datele folosind concepte inspirate din SQL, dar implementate într-un limbaj de programare funcțional.

Gestionarea bazelor de date:
- Crearea, ștergerea și selectarea tabelelor.
- Implementarea operațiunilor de JOIN între tabele.

Filtrarea datelor:
- Definirea unor condiții de filtrare flexibile.
- Suport pentru operații logice: AND, OR, NOT, EQUAL.

Interfață SQL-like:
- DSL pentru operațiuni ca CREATE TABLE, DROP TABLE, SELECT, JOIN.

Operațiuni pe tabele:
- Inserarea, ștergerea, sortarea, selectarea coloanelor.
- Manipularea rândurilor ca map-uri.

Funcții specifice:
- Exemple de interogări specifice (de exemplu: eliminarea rândurilor cu Jack Sparrow, inserarea și sortarea rândurilor).
